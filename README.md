# RideWave UI Simulation

UI simulation for Rider + Driver + Admin workflows. A safe public demo repository with no real keys or credentials.

## Overview

RideWave simulates a ride-hailing platform's user interface, demonstrating three distinct user flows: Rider booking, Driver dispatch, and Admin fleet management. Built for portfolio demonstration and UI/UX experimentation.

## Features

- **Rider Experience:** Request a ride, track ETA, fare estimation, ride history
- **Driver Experience:** Trip acceptance, navigation mock, earnings dashboard
- **Admin Panel:** Fleet overview, rider/driver analytics, trip monitoring

## Tech Stack

- **UI Framework:** React (create-react-app or Vite)
- **State Management:** Redux / Context API
- **Styling:** CSS Modules / Tailwind CSS
- **Mock Data:** JSON server or hardcoded fixtures

## Getting Started

```bash
git clone https://github.com/Raphasha27/ridewave-ui-simulation.git
cd ridewave-ui-simulation
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).
